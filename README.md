## 技术栈
vue2 + vue-rotuer2 + webpack + ES6/7 + less + flex + svg + postcss

## 项目运行
```
克隆，或者直接下载
git clone https://github.com/SealUI/vue-sealui-m.git

进入文件夹
cd vue-sealui-m

安装依赖
npm install
```

## 编译环境
```

开启本地服务器
npm run dev

访问 http://localhost:8000
```

## 线上版本
```
npm run build
```

## 浏览器支持
现代浏览器和Internet Explorer 9 +
